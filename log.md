## Git log

**git log** - это команда для просмотра истории коммитов в репозитории.

### Описание

Отображает историю коммитов в обратном хронологическом порядке, показывая хеши, авторов, даты и сообщения коммитов, с возможностью фильтрации, поиска и форматирования вывода.


Простой вывод истории:

```bash=
git log
```
Показать изменения в файлах:

```bash=
git log --stat
```
Показать полный diff каждого коммита:

```bash=
git log -p
```
